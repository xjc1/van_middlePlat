# OperateBar `系列操作`

当表格每一条记录的操作按钮比较多的情况下，就应该首先考虑使用该组件。

### 在表格中使用

<Preview>
  <Story name="inTable">
    <TTable
      dataSource={data.list}
      columns={[
        {
          title: '姓名',
          dataIndex: 'name'
        },
        {
          title: '年龄',
          dataIndex: 'age'
        },
        {
          title: '操作',
          align: 'center',
          render: (text, record) => (
            <OperateBar
              more={
                <>
                  <OperateBar.Button
                    onClick={() => console.log('编辑', record)}
                  >
                    编辑
                  </OperateBar.Button>
                  <OperateBar.Button
                    danger
                    onClick={() => console.log('删除', record)}
                  >
                    删除
                  </OperateBar.Button>
                </>
              }
            >
              <OperateBar.Button
                onClick={() => console.log('查看', record)}
              >
                查看
              </OperateBar.Button>
            </OperateBar>
          )
        }
      ]}
      pagination={{
        defaultPageSize: 5
      }}
      rowKey="key"
    />
  </Story>
</Preview>
