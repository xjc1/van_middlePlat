# UploadImage `图片上传控件`

基于 `antd` 的 `Upload` 组件实现的表单自定义控件，这里仅仅是封装了其组件属性 `listType="picture-card"` 的情况。

通过此组件上传的图片都会处理成 `Base64` 的字符串。
